1. Introduction

DiVAS (Distributed Video Archiving System) is an open-source NVR solution for a large scale video installation. Currently, it only allows the user to download mpeg4 archives without the ability for interactive playback. Future versions are likely to include support for alternate codecs and container formats as well as interactive viewing of the archives.

2. Requirements

DiVAS is a Java application distributed via Java WebStart. You must have Java 1.5 or higher installed to run the application. Get the Java Runtime Environment.
