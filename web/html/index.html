Introduction

HyperStream is a viewer for Mn/DOT streaming video of traffic cameras. It is currently being made available to organizations that report on or respond to situations on the metro area freeway system. There are currently more than 300 cameras located in the metro area which are available for viewing.

Installation

Install the Java Runtime Environment 1.5 or higher. It is available from Sun Microsystems This will automatically install Java WebStart which is needed to launch HyperStream. This must be done by someone with local admin privileges.

Running

In order to run HyperStream, you must have an account. Once your account is setup, you can view streaming video on our streaming video website

Account Maintenance

You can change your password by using the password change form.
